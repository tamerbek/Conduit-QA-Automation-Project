Проект по автоматизации тестирования на Python. Покрывает два направления: API Testing и Web UI Testing.

Документация API: https://api.realworld.show/docs

**Чеклист проверок:**

**Проверки UI**

**Страница регистрации**
- Регистрация с валидными данными
- Проверка кнопки "Have an account?"

**Страница логина**
- Логин с валидными данными
- Проверка кнопки "Need an account?"

**Проверки API:**

**Регистрация пользователя (post, /users):**

- Регистрация с валидными данными
- Пустые / отсутствующие поля
  - Без username
  - Без email
  - Без password
  - Все поля пустые строки
  - Пустой body
  - Body без обёртки user
- Дубликаты
  - Повторный username (api позволяет использовать дубликаты)
  - Повторный email (api позволяет использовать дубликаты)
- Валидация email
  - email длиной 100 симоволов
  - email длиной 101 символ
-Валидация username
  - username длиной 60 симоволов
  - username длиной 61 символ
- Валидация password
  - password длиной 60 символов
  - password длиной 61 символ
- Безопасность
  - SQL инъекция в пароле

**Логин пользователя (post, /users/login):**

- Логин с валидными данными
- Путые / отсутствующие поля
  - Неверный пароль
  - Без пароля
  - Без username
  - Без username и пароля
  - Пустой body
  - Body без обёртки user
- Безопасность
  - SQL инъекция в пароле
